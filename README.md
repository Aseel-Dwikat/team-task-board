## Overview

**Team Task Board** is a simplified task management app inspired by Jira. It supports creating, editing, deleting, searching, filtering, and sorting tasks — all without a single external dependency at runtime. Every change is saved instantly to `localStorage`, so your board survives a page refresh.

The project was built as a frontend assignment focused on writing clean, modular, strictly-typed TypeScript without relying on frameworks or CSS libraries.

## Features

| Category | Details |
|---|---|
| ✏️ **Create & Edit** | One form handles both — switches to "Save Changes" mode automatically when editing an existing task |
| 🗑️ **Delete** | Custom-built confirmation modal (not the native browser popup), dismissible via button, backdrop click, or `Esc` |
| 🔄 **Status Updates** | Change a task's status directly from its card — no need to open the edit form |
| 🔍 **Live Search** | Instantly filters tasks by title or assignee as you type |
| 🎚️ **Filters** | By status and priority, fully combinable with search |
| ↕️ **Sorting** | By due date, priority, or creation date — works alongside active filters and search |
| ✅ **Validation** | Required fields are visually flagged before submission is allowed |
| 🗂️ **Empty States** | Distinct messages for "no tasks yet" vs. "no results for your filters" |
| 📱 **Responsive** | Built with CSS Grid & Flexbox — no horizontal scroll on any screen size |
| 🎬 **Animations** | Staggered card entrances, hover/focus feedback, animated delete modal — all respecting `prefers-reduced-motion` |
| 💾 **Persistence** | Every create, edit, delete, and status change is saved to `localStorage` immediately |

## Tech Stack

- **TypeScript** — strict mode enabled, fully modular, zero `any`
- **Vanilla CSS** — Grid, Flexbox, custom keyframe animations
- **Vanilla HTML** — semantic, no templating engine
- No frameworks, no CSS libraries, no bundler — just the TypeScript compiler

## Project Structure

```
task-board/
├── index.html            # App markup + delete confirmation modal
├── main.css              # Styling and animations
├── tsconfig.json         # TypeScript compiler configuration
├── src/
│   ├── task.ts            # Task model & types
│   ├── storage.ts         # localStorage read/write helpers
│   ├── render.ts           # Renders task cards + animation helpers
│   ├── modal.ts             # Custom confirm-dialog logic
│   └── app.ts                # Main app logic: form handling, filters, sorting
└── dist/                  # Compiled JavaScript output (generated by tsc)
```

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org) (for the TypeScript compiler)

### 1. Install dependencies
```bash
npm install
```

### 2. Build
```bash
npx tsc -p tsconfig.json
```
Compiles everything in `src/*.ts` into `dist/*.js`.

### 3. Run
Open `index.html` directly in your browser — no server required.

> 💡 Using an extension like VS Code's **Live Server** is recommended over opening the file directly, to avoid `file://` restrictions with ES Modules.

## Development Notes

- The `dist/` folder **is** committed to the repository (not `.gitignore`d), since the app loads the compiled JS directly with no bundler or build server in front of it.
- `node_modules/` is excluded via `.gitignore` — run `npm install` after cloning to regenerate it.
- Run `npx tsc -p tsconfig.json` after any change to a `.ts` file, or use `npx tsc -p tsconfig.json --watch` to rebuild automatically on save.
